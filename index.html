Project

During the regular season of the NBA there are all up 1230 games. Now this is too many games for one person to watch, so I would want to create a method that will help me decide what games are most worth me watching. This project would take the form of a website that analyses the play of these of key players and recommends the games based on how well they played compared to their averages. I am a big fan of watching fellow Australians play in the NBA so most of the players that I would choose to be analysed would be Australia players who play for a variety of different teams.

This project would help me save loads of time in deciding what games I should watch over the course of the NBA season. This will be created using a mixture of html and the NBA-API and endpoint. The NBA-API allows live stats to be streamed to a website or app.

One feature of the website that is vital, is that no results or stats from the game should be shown. Instead an indicator should be shown based on the watchability of the game based on the stats of the players I have selected. This means that I won’t have to deal with any spoilers when accessing the website. A spoiler being accidently shown will dramatically reduce the watchability of the game, if the results of the game is already known

The way the watchability of the game should be determined could be done so through a variety of factors. First the average points per game and assists per game of a player can be compared with the single game, if there is a significant increase in production of that player in the game, then it should be recommended.

If within the last 2 minutes of game time the score is no bigger than 10 points between the two teams. Close games are a good game, and the games should be extra recommended if the game goes down to the wire.

If there are two or more key players in the same game, then it should be recommended. If two or more players are playing against each other then the game should almost automatically be highly recommended. This is because if Ben Simmons and Patty Mills are versing each other I would want to see it.

The players that a featured in the ideal games to be watched should be easily added and changed by the end user without too much hassle. This will allow for anyone to have a personalised version of the website better allowing more people to save time in deciding what NBA games they want to watch.

I don’t have any experience in making these types of projects, but everyone must start somewhere. Also, this project will help me in my progress towards my ideal job. It is a perfect building point for the skills needed to be acquired to work at my ideal job. The combination of NBA and web development is basically the perfect duo that could be possible.

If the project is successful, the best NBA game for each day will be recommended allowing the hasn’t user to save time and stress in deciding what the best game to watch is. A spoiler free game recommender is a very useful tool, I could not find any similar products on the market that allows this type of recommendation without showing stats. This allows people with busy lives to have the best NBA experience they can possibly have without spending too much time.
